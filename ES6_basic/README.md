# ES6 Basic

Exercises for ES6 basics:

- const and let
- block scope
- arrow functions
- default parameters
- rest/spread syntax
- template literals
- object shorthand and computed properties
- method properties
- for...of loops
- simple report objects

## Setup

1. Install dependencies:

```bash
npm install
```

2. Lint:

```bash
npm run lint
```

3. Run tests:

```bash
npm run test
```
